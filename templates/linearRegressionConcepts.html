Linear Regression

Basic concepts, variables, regression line, slope and intercept explained with real examples.

DEFINITION

What is Linear Regression?

Linear Regression is a supervised machine learning technique used to predict a continuous numeric value based on one or more input variables. It finds the best straight line that fits the data, minimizing the error between predicted and actual values.

It assumes a linear relationship between the input (independent variable) and the output (dependent variable). The more study hours, the higher the grade. The more ad investment, the higher the sales.

EQUATION

The Linear Regression Formula

y = ax + b

Simple Linear Regression

y — Dependent variable

The value we want to predict. Example: final grade, house price, sales.

x — Independent variable

The input data we use to make the prediction. Example: study hours.

a — Slope

How much y changes for each unit increase in x. The steepness of the line.

b — Intercept

The value of y when x = 0. Where the line crosses the Y axis.

VARIABLES

Independent vs Dependent Variables

Independent Variable (X)

The input. The variable we control or observe. It causes the change.

Example: Study hours
Dependent Variable (Y)

The output. The variable we want to predict. It depends on X.

Example: Final grade

REGRESSION LINE

The Regression Line

The regression line is the best fit straight line drawn through the data points. It minimizes the sum of squared distances between each data point and the line — this is called Ordinary Least Squares (OLS).

Slope (a)

Indicates how steep the line is. A positive slope means that as X increases, Y also increases. In our model: for each extra study hour, the grade increases by approximately 0.18 points.

Intercept (b)

The point where the line crosses the Y axis (when X = 0). In our model: a student who studies 0 hours would get approximately 1.2 as a base grade.
