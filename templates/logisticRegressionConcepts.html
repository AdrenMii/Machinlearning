DEFINITION

What is Logistic Regression?

Despite its name, Logistic Regression is a supervised machine learning algorithm used for Classification. It predicts the probability that an input belongs to a specific category (0 or 1, Yes or No, True or False).

It maps any real-valued number into a value between 0 and 1 using the Logit function. It is widely used for medical diagnosis, spam detection, and financial fraud prevention.

THE SIGMOID FUNCTION

The Mathematical Core

a = 1 / (1 + exp(-z))

Logistic/Sigmoid Function

z — Input Linear Model

Represented as $z = w \cdot x + b$, where $w$ are weights and $b$ is bias.

a — Output Probability

The result is a value between 0 and 1. If $a > 0.5$, the model predicts class 1.
